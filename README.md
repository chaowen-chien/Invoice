# Invoice #

這是一個發票對獎小工具。輸入發票號碼末三碼，程式將以訊息與音效提示是否中獎。

![](https://raw.githubusercontent.com/chaowen-chien/Invoice/master/Image/Example.gif "發票對獎範例")

若有需要可以修改 Invoice.ini 檔案，自行輸入發票對獎號碼。
